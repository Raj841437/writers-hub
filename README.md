# 🌙 Writers Hub

A social platform where words find a home.

Writers Hub is a web application built with Django that allows users to express their thoughts through poems and stories, connect with others, and have meaningful conversations.

---

## ✨ Features

* ✍️ Create and share poems or stories
* ❤️ Like posts
* 💬 Comment on posts
* 👤 User authentication (Signup/Login)
* 👥 Friend request system
* 💬 Private chat between users
* 🌙 Dark themed emotional UI

---

## 🛠️ Tech Stack

* Backend: Python (Django)
* Frontend: HTML, CSS
* Database: SQLite
* Version Control: Git & GitHub

---

## 🚀 Getting Started

### 1. Clone the repository

git clone https://github.com/Raj841437/writers-hub.git

### 2. Navigate into the project

cd writershub

### 3. Install dependencies

pip install django

### 4. Run migrations

python manage.py makemigrations
python manage.py migrate

### 5. Run server

python manage.py runserver

### 6. Open in browser

http://127.0.0.1:8000/
